이 작업은 배민 배달처리비 데이터를 기반으로 매일 또는 주기적으로 수행하는 정산 보조 프로세스임.

**1. 다운로드 및 로그인 절차**
- **스크립트:** `~/send_batch/baemin_login.sh`
- **수행 내용:** Chrome을 통해 배민 관리자 사이트에 로그인(2FA 자동 처리) 후 `배달처리비 다운로드` 페이지로 이동.
- **주의 사항:** 다운로드 시 사유 입력창에 "정산"을 입력해야 함.

**2. 비밀번호 제거 (복호화)**
- **비밀번호:** 계정 ID (`flatform845`)
- **도구:** `msoffcrypto-tool`
- **방법:** `project_excel_password_removal.md`에 정의된 로직을 사용하여 `.xlsx` 파일의 암호를 해제함.

**3. 정산 요약 시트 생성 (Python 로직)**
- **입력:** 복호화된 배달처리비 Excel 파일 + `~/send_batch/riders.json` (연락처 매핑용)
- **수행 내용:**
    1. `라이더명`, `User ID`, `배달처리비` 컬럼을 추출하여 라이더별 건수와 합계를 계산.
    2. `riders.json`에서 `(이름, User ID)`를 키로 사용하여 연락처(휴대폰번호)를 매칭.
    3. `정산요약` 시트를 신규 생성하여 [라이더명, 아이디, 연락처, 배달건수, 합계금액] 순으로 기록.
    4. 금액 누락 건 발생 시 사용자에게 즉시 보고.

**4. 주요 경로**
- **마스터 데이터:** `/Users/settlment/send_batch/riders.json`
- **로그인 환경변수:** `/Users/settlment/Library/CloudStorage/OneDrive-개인/프로젝트/telegram_claude_bot/.env`

**Why:** 매일 발생하는 정산 데이터를 수동으로 정리하는 번거로움을 줄이고, 연락처 매핑 오류를 방지하기 위함.
**How to apply:** 사용자가 "배민 배달처리비 다운로드 및 정산"을 요청하면 위 프로세스를 순차적으로 실행함.
